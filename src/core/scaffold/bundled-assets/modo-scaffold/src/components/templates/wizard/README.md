# Wizard

分步导航式的表单向导页面模板。适用于需要分步骤完成的复杂任务，能够引导用户按顺序完成操作，并清晰展示当前进度。

## 使用场景

-   配置流程复杂的资源创建（如数据接入、指标模型配置等）。
-   需要分拆展现的注册、发布或其他多图层表单任务。
-   将长卷轴表单切分为多个短小的步骤，以降低业务人员的认知操作负担。

## 页面结构

该模板页面遵循 MODO 规范分为以下几个重要模块：

1.  **顶部步骤条 (📍 Header Steps)** 
    - 结合了深色/浅色背景及特定的外边距。
    - 使用定制后的 `ModoSteps` 展现各个分段状态（进行中、未完成、已完成），保持宽度居中。

2.  **表单内容区 (📝 Content Scroll Area)**
    - 内容区采用 `flex-1 overflow-y-auto` 的布局使其支持独立上下滚动，而不带走上方步骤条。
    - 使用了 Ant Design 原生 `Form` 及响应式 `Row/Col` 组件进行两栏或全栏混排式布局。

3.  **底部操作栏 (⚙️ Footer Tools)**
    - 吸附在页面最下方的操作工具条。
    - 提供全局“保存”、“上一步”、“下一步”、“提交/取消”等操作，通常与步骤所处的位置联动。

## 核心组件使用

-   `ModoPage`: 使用带有 `!p-0` 及 `!shadow-none` 清除默认的页间距和白色阴影卡片以满足吸附设计。
-   `ModoSteps`: 展示进度状态并负责样式对齐。
-   `ModoSelect`, `ModoInput`: MODO 组件库标准表单控件。
